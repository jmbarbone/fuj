# `{fuj}` options

Options uses for `{fuj}` functions.

## Usage

``` r
op.fuj
```

## Format

An object of class `list` of length 5.

## Examples

``` r
names(op.fuj)
#> [1] "fuj.vap.progress"       "fuj.vap.indexed_errors" "fuj.verbose"           
#> [4] "fuj.verbose.fill"       "fuj.verbose.label"     
op.fuj
#> $fuj.vap.progress
#> [1] FALSE
#> 
#> $fuj.vap.indexed_errors
#> [1] FALSE
#> 
#> $fuj.verbose
#> NULL
#> 
#> $fuj.verbose.fill
#> [1] FALSE
#> 
#> $fuj.verbose.label
#> [1] "verbose: "
#> 
```
