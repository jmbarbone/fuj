# Arithmetic wrappers

Arithmetic wrappers

## Value

See [base::Arithmetic](https://rdrr.io/r/base/Arithmetic.html)

## Examples

``` r
        add(7, 2) # +
#> [1] 9
   subtract(7, 2) # -
#> [1] 5
   multiply(7, 2) # *
#> [1] 14
     divide(7, 2) # /
#> [1] 3.5
raise_power(7, 2) # ^
#> [1] 49
  remainder(7, 2) # %%
#> [1] 1
 divide_int(7, 2) # %/%
#> [1] 3
```
